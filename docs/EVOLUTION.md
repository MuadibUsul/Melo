# 演进预留接口

本文档定义后续版本（v1.1+）预留的架构接口，当前不实现具体逻辑，仅保留接口契约。

## HLS 自适应流 (HTTP Live Streaming)

```
// packages/contracts/src/media.ts (proposed)
interface HlsManifest {
  url: string;
  segments: { uri: string; duration: number }[];
  bitrates: number[];
}

// API endpoint (reserved)
// GET /api/v1/tracks/:id/hls → HlsManifest
```

**用途**: 长音频自适应码率流式传输，目前使用签名 URL 直链。

## Meilisearch 全文搜索

```
// packages/contracts/src/search.ts (proposed)
interface SearchIndex {
  tracks: "title" | "creator.displayName" | "tags" | "lyrics";
  users: "displayName" | "email";
}

// API endpoint (reserved)
// GET /api/v1/search?q=&index=tracks&filters=...
```

**用途**: 替代当前 PostgreSQL ILIKE，提供拼音搜索、同义词、相关性排序。当前使用 PG 内置搜索。

## ClickHouse 分析

```
// packages/contracts/src/analytics.ts (proposed)
interface PlayAnalytics {
  trackId: string; plays: number; uniqueListeners: number;
  avgPlayDuration: number; completionRate: number;
  period: "hourly" | "daily" | "weekly";
}

// API endpoint (reserved, admin only)
// GET /api/v1/admin/analytics/plays?period=daily
```

**用途**: 高基数播放事件分析、实时看板。当前使用 PostgreSQL + Redis sorted sets。

## 推荐系统

```
// packages/contracts/src/recommendation.ts (proposed)
interface RecommendationRequest {
  trackId?: string; // seed track
  userId?: string;
  limit?: number;
}

interface RecommendationResult {
  items: { trackId: string; score: number; reason: string }[];
}

// API endpoint (reserved)
// GET /api/v1/recommendations?seed=trackId
```

**用途**: 基于协作过滤或内容的推荐。当前使用排行榜 + 流派筛选。
